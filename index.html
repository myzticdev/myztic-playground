Common questions

Good to know.

Is Myztic Playground really free?

Yes. It is free to use, with no account, trial, or payment details required.

Where is my code saved?

Your current project is saved in your browser’s local storage. It is not uploaded to a Myztic server.

Can I use libraries or external APIs?

No. Network access is blocked inside the preview to keep the sandbox isolated and predictable.

Is it safe to run JavaScript?

User code runs in a restricted iframe without same-origin access, forms, popups, downloads, or top-level navigation. Read the sandbox details.
